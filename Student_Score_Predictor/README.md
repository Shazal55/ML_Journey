# 🎓 Student Performance Predictor

A machine learning project that predicts a student's Performance Index using Multiple Linear Regression implemented from scratch with NumPy.

## Project Overview

This project uses academic and lifestyle factors to predict a student's performance score. The model is trained using Gradient Descent without relying on machine learning libraries such as Scikit-Learn.

## Dataset Features

- Hours Studied
- Previous Scores
- Extracurricular Activities
- Sleep Hours
- Sample Question Papers Practiced

### Target Variable

- Performance Index

## Workflow

1. Load and analyze the dataset
2. Encode categorical variables
3. Split data into training and testing sets
4. Apply feature scaling
5. Train a Multiple Linear Regression model
6. Optimize parameters using Gradient Descent
7. Evaluate model performance using Mean Squared Error (MSE)

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib

## Model Implementation

The model was implemented completely from scratch using NumPy, including:

- Vectorized predictions
- Mean Squared Error (MSE)
- Gradient computation
- Gradient Descent optimization
- Parameter updates

## Results

Training Loss: ~4.14

Test Loss: ~4.18

The close training and testing losses indicate that the model generalizes well to unseen data.
